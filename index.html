Projects

  • Pokebank

    A project built at Hack the North 2019. A savings app created using React Native and JavaScript which incentivized young people to put money into their savings account. Implemented additional details like a login database using Firebase and a voice command system using Alexa and Voiceflow.

  • Glasswings

    A project built at Hack the 6ix 2019. An insurance web app built with Python. Used crowd sourced environmental data and insurance price data to estimate how much customers should pay for insurance. Used AWS Sagemaker to create a machine learning model.

    A Discord bot created to greet server users, send them a manual, as well as the option to play games with the users. Used node to integrate with Discord as well as JavaScript.

    A Java game inspired by foddy.net's "Too Many Ninjas!" game. Designed the game using object oriented programming concepts and design patterns.
